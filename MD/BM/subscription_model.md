# PULSE 구독 모델 전략 (Subscription Model Strategy)

## 1. 전략 개요 (Strategy Overview)
PULSE의 가격 전략은 **"낮은 진입 장벽(Low Barrier)"**과 **"고부가가치 제안(High Value-Add)"**에 초점을 맞춥니다.
소상공인 사장님들은 고정비 지출에 민감하지만, **"매출에 직결되는 도구"**에는 기꺼이 지갑을 엽니다.

## 2. 요금제 구조 (Tier Structure)

### **🌱 PULSE Basic (무료 / Free)**
**"장사비서" - 사장님의 기본 운영을 돕는 비서**
*   **타겟**: 마케팅이 낯설고 비용 지출을 꺼리는 소규모 1인 사장님
*   **핵심 가치**: 부담 없는 매장 현황 파악 및 "마케팅의 맛" 경험
*   **제공 기능**:
    *   **매장 대시보드**: 매출/방문객 추이, 날씨/시즌 기반 운영 팁 제공
    *   **우리 동네 요약**: 동네 주요 연령대/성별 데이터 간단 요약
    *   **홍보 영상 만들기**: **월 1회 무료 제작** (워터마크 포함)
    *   **커뮤니티**: 사장님 커뮤니티 "장사 노하우" 열람 권한

### **📈 PULSE Growth (월 19,000원 ~ 29,000원)**
**"단골 메이커" - 단골 손님을 만들고 관리하는 단계**
*   **타겟**: 매출 성장을 위해 능동적으로 움직이는 사장님 (Active User)
*   **핵심 가치**: 정교한 손님 분석을 통한 "타겟 마케팅" 실현
*   **제공 기능 (Basic +)**:
    *   **손님 마음 읽기 (무제한)**: 상세 페르소나 분석, 고객 여정 지도(CJM) 열람
    *   **AI 마케팅 코치**: "오늘 뭐 팔지?" AI 챗봇 무제한 질문/답변
    *   **리뷰 관리**: AI 리뷰 답글 자동 생성 (감정 분석 포함)
    *   **홍보 영상 만들기**: **월 5회 제작** (FHD 화질, 워터마크 제거)
    *   **카카오 알림톡**: 단골 손님 대상 쿠폰 발송 기능 (발송 건당 별도 과금)

### **🚀 PULSE Pro (월 49,000원 ~ 69,000원)**
**"홍보 마스터" - 공격적인 확장을 원하는 사장님**
*   **타겟**: 2호점 확장을 계획하거나 지역 내 1등이 목표인 사장님
*   **핵심 가치**: 외부 홍보 확장 및 전문가 연결 (Offline Connection)
*   **제공 기능 (Growth +)**:
    *   **인플루언서 매칭**: 지역 푸드 크리에이터/인플루언서 매칭 지원 (매칭 수수료 0%)
    *   **홍보 영상 만들기 (무제한)**: 횟수 제한 없음, **4K/Pro 퀄리티** 지원
    *   **경쟁사 분석**: 경쟁 업체(유사 업종) 상세 성과 분석 리포트 제공
    *   **다점포 관리**: 2개 이상의 매장을 통합 관리할 수 있는 멀티 대시보드

## 3. 기능 비교표 (Feature Matrix)

| 구분 | Basic (무료) | Growth (실속형) | Pro (프리미엄) |
| :--- | :---: | :---: | :---: |
| **대시보드** | 기본 (매출/날씨) | 상세 (고객 패턴) | 통합 (다점포) |
| **손님 분석** | 지역 요약 | **상세 페르소나** | 상세 + 경쟁사 |
| **AI 챗봇** | 일 3회 제한 | **무제한** | 무제한 |
| **영상 제작** | 월 1회 (Watermark) | **월 5회 (FHD)** | **무제한 (4K)** |
| **리뷰 관리** | 뷰어 제공 | **AI 답글 생성** | AI 답글 + 자동 등록 |
| **인플루언서** | X | X | **매칭 지원** |
| **월 요금** | **0원** | **29,000원** | **59,000원** |

## 4. 왜 이 모델인가? (Why This Model?)
1.  **Zero Risk Start (무위험 진입)**: `Basic` 단계에서 일단 매일 들어오게 만들어 습관(Lock-in)을 형성합니다.
2.  **Visual Hook (시각적 유인)**: `Growth`로 전환시키는 핵심 트리거는 **"영상 제작(Video Creator)"**입니다. 무료로 1개 만들어보고 퀄리티에 만족하면, "하나 더 만들고 싶은데?" 할 때가 결제 타이밍입니다.
3.  **Revenue Scale (수익 확장)**: `Pro`는 단순한 SW 툴이 아니라 **"인플루언서"**라는 오프라인 가치를 제공하므로, 5~6만 원대의 가격 저항감을 획기적으로 낮출 수 있습니다.
