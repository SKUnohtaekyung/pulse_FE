📚 웹 인터랙션 백과사전 (Web Interaction Dictionary)

이 문서는 Interactive_Scroll_Demo_All_In_One.html에 구현된 19가지(Intro 포함) 핵심 애니메이션과 글로벌 효과들을 정리한 기술 명세서입니다. AI나 개발자에게 작업을 요청할 때 **'정식 명칭(English Keyword)'**을 사용하면 가장 정확한 결과를 얻을 수 있습니다.

🌍 글로벌 효과 (Global Effects)

페이지 전체에 적용되어 사용자 경험(UX)의 질을 높이는 기반 기술입니다.

정식 명칭 (Keyword)

한글 명칭

설명 및 요청 가이드

Lenis Smooth Scroll

부드러운 스크롤

마우스 휠의 뚝뚝 끊기는 움직임을 관성(Inertia)이 적용된 부드러운 움직임으로 변환합니다. 모든 고급 인터랙션의 필수 기반입니다.

Cinematic Preloader

시네마틱 프리로더

사이트 리소스가 로딩될 때까지 진행률을 보여주다가, 완료되면 검은 막(Curtain)이 걷히듯 콘텐츠를 등장시킵니다.

Custom Magnetic Cursor

자석 커서

기본 마우스 커서를 숨기고 커스텀 원형 커서를 사용합니다. 버튼이나 특정 요소 위에서는 자석처럼 달라붙거나(Magnetic) 크기가 커지는 반응을 합니다.

Grain Noise Overlay

그레인 노이즈

화면 전체에 옅은 TV 노이즈 질감을 덮어씌워, 디지털의 차가운 느낌을 줄이고 필름 영화 같은 아날로그 감성을 더합니다.

Floating Dock Nav

플로팅 독 내비게이션

macOS의 독(Dock)처럼 하단에 떠 있는 메뉴바입니다. 현재 스크롤 위치를 표시하고 클릭 시 해당 섹션으로 부드럽게 이동합니다.

🎨 섹션별 인터랙션 (Section Interactions)

1. 오프닝 & 분위기 (Intro & Mood)

1. Mask Reveal (Clip-Path Animation)

한글 명칭: 마스크 리빌

설명: 검은 화면 속 작은 원(Mask)이 스크롤에 따라 점점 커지며 배경 이미지가 드러나는 기법입니다.

활용: 브랜드 오프닝, 극적인 인트로 연출.

2. Theme Transition (Background Color Swap)

한글 명칭: 테마 반전 / 다크모드 전환

설명: 스크롤이 특정 섹션에 도달하면 body의 배경색과 글자색이 부드럽게 반전(Dark ↔ Light)됩니다.

활용: 긴 스크롤 페이지에서 분위기를 환기시키거나 강조하고 싶은 섹션에 사용.

2. 타이포그래피 & 가이드 (Typography & Guide)

3. 3D Rolling Text (Perspective Rotate)

한글 명칭: 3D 롤링 텍스트

설명: 텍스트가 원통형으로 회전하듯 X축으로 기울어지며 등장합니다. perspective 속성을 활용해 웅장한 입체감을 줍니다.

활용: 영화 예고편 스타일의 헤드라인, 임팩트 있는 키워드 나열.

4. SVG Path Drawing (Stroke Dashoffset)

한글 명칭: 패스 드로잉 / 라인 드로잉

설명: 스크롤을 내리는 동안 SVG 라인이 펜으로 그리는 것처럼 실시간으로 그려집니다.

활용: 사용자의 시선을 다음 콘텐츠로 유도하거나, 제품의 연결 선을 표현할 때.

9. Text Reveal (Background Clip)

한글 명칭: 텍스트 리빌 / 리딩 프로그레스

설명: 텍스트의 색상이 흐린 색에서 진한 색으로, 사용자가 읽는 속도(스크롤)에 맞춰 차오릅니다.

활용: 긴 문장이나 선언문을 사용자가 꼼꼼히 읽게 유도할 때.

14. Split Text Stagger

한글 명칭: 스플릿 텍스트 등장 효과

설명: 텍스트를 단어(Word)나 글자(Char) 단위로 쪼개고(Split), 마스킹된 영역 아래에서 시차(Stagger)를 두고 솟아오르게 합니다.

활용: [랜딩 페이지 필수] 메인 비주얼(Hero)의 제목 등장 시 가장 세련된 표준 기법.

18. RGB Kinetic Typography

한글 명칭: RGB 키네틱 / 글리치 텍스트

설명: 스크롤 속도가 빠를수록 텍스트의 Red, Green, Blue 채널이 분리되어 흔들리는(Glitch) 효과를 냅니다.

활용: 속도감, 하이테크, 사이버펑크 분위기 연출.

3. 이미지 & 갤러리 (Image & Gallery)

5. Explode Gallery (Scatter Effect)

한글 명칭: 익스플로드 갤러리

설명: 중앙에 겹쳐져 있던 이미지들이 스크롤과 함께 사방으로 흩어지며(Explode) 공간감을 형성합니다.

활용: 포트폴리오 모음, 제품의 구성품 전개도.

6. Velocity Skew (Physics Distortion)

한글 명칭: 벨로시티 스큐 / 관성 왜곡

설명: 스크롤 속도(Velocity)에 비례하여 이미지가 젤리처럼 찌그러지거나(Skew) 기울어집니다.

활용: 정적인 갤러리에 물리적인 생동감과 속도감을 부여할 때.

8. Pin & Scale (Immersive Zoom)

한글 명칭: 핀 & 스케일 / 딥 줌

설명: 스크롤을 해도 화면이 고정(Pin)된 상태에서, 이미지가 화면 가득 확대(Scale Up)되어 사용자를 빨아들이는 듯한 느낌을 줍니다.

활용: 제품의 디테일 샷, 새로운 챕터로의 전환.

15. Accordion Gallery

한글 명칭: 아코디언 갤러리

설명: 여러 이미지가 가로로 나열되어 있고, 마우스를 올린 이미지만 부드럽게 확장되며 나머지는 축소됩니다.

활용: [랜딩 페이지 필수] 제한된 공간에 서비스의 주요 기능(Features)을 요약해서 보여줄 때.

16. Hover Image Reveal

한글 명칭: 호버 이미지 리빌 / 마우스 팔로우 이미지

설명: 텍스트 리스트에 마우스를 올리면, 해당 항목과 관련된 이미지가 마우스 커서를 따라다니며 나타납니다.

활용: [랜딩 페이지 필수] 작품 목록(List), 목차, 메뉴 등에서 시각적 정보를 함께 전달할 때.

4. 마우스 인터랙션 & 3D (Mouse & 3D)

7. Holo-Card (3D Tilt Effect)

한글 명칭: 홀로그램 카드 / 3D 틸트

설명: 마우스 위치에 따라 카드가 3D로 기울어지며, 마우스 반대 방향에서 빛(Glare)이 반사되는 효과를 시뮬레이션합니다.

활용: 프리미엄 멤버십 카드, 신용카드, 중요 배너.

12. Gravity Physics (Physics Engine)

한글 명칭: 중력 물리 엔진

설명: 2D 캔버스 상에서 공들이 중력, 마찰력, 탄성 계수를 가지고 움직입니다. 마우스로 공을 치거나 던질 수 있습니다.

활용: 사용자의 체류 시간을 늘리기 위한 '놀이(Playground)' 요소.

17. Interactive Water Ripple

한글 명칭: 수면 파동 / 유체 시뮬레이션

설명: 마우스가 지나가는 자리에 물리적으로 계산된 물결 파동이 일어납니다. 반사와 굴절을 표현하여 매우 사실적입니다.

활용: 브랜드의 신비로움, 청량함, 부드러움을 강조할 때 배경 효과.

5. 생성 예술 & 고급 기술 (Generative Art)

10. Particle Text Disperse

한글 명칭: 파티클 텍스트 / 픽셀 분산

설명: 텍스트의 픽셀 데이터를 읽어와 수천 개의 입자(Particle)로 변환합니다. 마우스가 다가오면 입자들이 도망가거나 흩어집니다.

활용: "Touch Me"와 같은 인터랙션 유도, 기술력 과시용 헤더.

11. Constellation Network

한글 명칭: 별자리 네트워크

설명: 화면을 떠다니는 입자들끼리 일정 거리 이내로 가까워지면 선을 연결합니다. 마우스와도 연결되어 상호작용합니다.

활용: AI, 네트워크, 데이터, 연결성을 강조하는 테크 기업의 배경.

13. Generative Flow Field

한글 명칭: 플로우 필드 / 유동장

설명: 수많은 입자가 보이지 않는 힘의 흐름(Vector Field)을 따라 이동하며 궤적을 남깁니다. 유화 물감이 번지는 듯한 예술적인 느낌을 줍니다.

활용: 예술적이고 몽환적인 분위기가 필요한 디자인 에이전시나 전시 사이트.

💡 AI에게 요청하는 팁 (Pro Tip)

조합 요청: "Hero 섹션에는 Split Text Stagger를 쓰고, 배경에는 Interactive Water Ripple을 깔아줘." 처럼 구체적인 기술명을 조합하면 훨씬 퀄리티 높은 결과물을 얻을 수 있습니다.

라이브러리 명시: "이 효과들은 GSAP ScrollTrigger와 Lenis를 사용해서 구현해줘"라고 도구를 지정하면 코드의 안정성이 높아집니다.
