# PULSE Copy Rules

Use this reference for Korean-first UX writing inside PULSE screens.

## Voice

- Korean first.
- 존댓말 by default.
- Friendly, practical, not corporate.
- Avoid vague hype and technical jargon.
- Use emojis sparingly only when they improve comprehension.

## Copy Rules

- Button labels: verb-led and short.
- Error messages: state what happened, why it matters, and what to do next.
- Empty states: explain the missing data and next action.
- Success states: confirm completion and next step.
- Dashboard text: scan-friendly, avoid long paragraphs.
- AI-generated suggestions: distinguish recommendation from confirmed fact.

## Surface Tone

- Owner dashboard: immediate, practical, action-oriented.
- Influencer inbox: trust, conditions, response clarity.
- Review management: calm, editable, recovery-friendly.
- Promotion/video: progress, asset limits, retry clarity.
- Auth/onboarding: confidence, visible steps, no surprise data requests.

## Output Shape

- Final recommended copy.
- Alternatives only when choice is useful.
- Tone rationale.
- UI fit warnings for long Korean text.
