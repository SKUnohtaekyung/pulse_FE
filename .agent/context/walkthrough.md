# PULSE 2.0 (V2) 랜딩페이지 UX 리팩토링 및 시연 가이드

PULSE V2 랜딩페이지의 CVR 방어 & 화려한 인터랙션 요구사항을 반영한 **[Phase 4: 구현]** 및 **[Phase 5: 검증]** 이 모두 완료되었습니다. 기존 `/` 경로를 안전하게 보호하면서, 진보된 V2 구조를 확립했습니다.

## 🚀 확인 방법 (시연 가이드)

현재 실행 중인 로컬호스트(`http://localhost:5173`) 환경에서 다음 두 가지 방식으로 결과를 테스트하실 수 있습니다.

1. **기존 홈페이지를 통한 진입 (A/B Test UX)**
   - 브라우저에서 `http://localhost:5173`로 접속하여 기존 V1 버전을 둘러봅니다.
   - **우측 하단에 떠 있는 [PULSE 2.0 UX Preview] 플로팅 버튼** 또는 화면 최하단의 **안내 배너**를 클릭하면 V2로 즉시 부드럽게 라우팅됩니다.
2. **다이렉트 진입**
   - 브라우저 주소창에 `http://localhost:5173/v2` 를 직접 입력하여 V2 화면으로 진입합니다.

## ✅ 주요 구현 사항 (Rule of Thumb 100% 통과)

> [!TIP]
> 4. 기술 아키텍트(@/Planner) 및 프론트엔드 개발자(@/Frontend Developer) 관점에서의 방어 로직이 모두 적용되었습니다.

### 1️⃣ Hero 섹션: CVR 타격 최소화
*   **전환율 사수**: 0.5초 안에 단어별로 스스륵 솟아오르는(Split Text Stagger) 애니메이션을 넣어 화려함을 더하되, **버튼을 가리기 전 즉시 사라지는 마감 시간 제약**을 두었습니다. 진입 즉시 클릭(CTA) 시도가 가능합니다.

### 2️⃣ Problem 아코디언 매핑
*   **지체 없는 정보전달**: "어떻게 홍보할지 몰라서" 같은 카트에 커서를 올리면 `flex-grow` 트랜지션을 통해 영역이 넓어지며 명쾌한 타개책 카피가 노출됩니다. 무거운 `Pin` 속성(스크롤 하이재킹)을 완전히 제어하여, 바쁜 유저는 스크롤 한 번에 지나갈 수 있도록 제작했습니다.

### 3️⃣ 루프 안내: Hover Image Reveal과 Mobile Graceful 예외
*   **인지 부하 최소화**: `3-Loop(기능)` 영역 텍스트 위로 올리면 화면 빈 곳에 모바일 UI 스크린샷이 나타납니다.
*   > [!IMPORTANT]
    > **모바일 화면 성능 방어 완료**: 화면이 좁고 커서가 나타나지 않는 터치 기반 브라우저 해상도(< 768px)에서는 무거운 CSS 위치 계산 로직을 전부 폐기(= Graceful Degradation)하고, 정적 이미지 Fade-In만 작동하게 만들어 프레임 드랍(Jank) 현상을 차단했습니다.

### 4️⃣ PULSE Pro 수익 모델 전환
*   **명확한 다크 테마 역전**: 랜딩페이지 막바지, `PULSE Pro (인플루언서 매칭)` 기능이 소개될 때 밝았던 배경이 **프리미엄 딥 다크 블루** 컬러로 전환되며, 웅장한 `3D Rolling Text` 임팩트로 나타납니다.
*   이를 통해 과금에 대한 거부감을 '오직 극상위 매장들을 위한 고급 파트너십 초대'라는 심리적 위안으로 바꿉니다.
